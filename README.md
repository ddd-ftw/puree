# Pure

Lightweight framework for creating DDD / CQRS based apps with a pure domain model of plain old ruby objects (PORO's)

## Installation

Add this line to your application's Gemfile:

    gem 'pure'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install pure
